# Develop Environment
- OS: Windows 11 Pro
- IDE: Pycharm
- Pycharm plugin: Pycharm-Blender-Plugin(https://github.com/BlackStartx/PyCharm-Blender-Plugin)
- Blender Version: 4.0

Extra modules: 
- Fake bpy: https://github.com/nutti/fake-bpy-module (pip install fake-bpy-module-4.0)
- Numpy: pip install numpy
